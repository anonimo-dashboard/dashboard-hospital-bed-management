\# Dashboard Hospital Bed Management



Executive dashboard developed in Streamlit for exploratory analysis and forecasting of hospital bed occupancy using synthetic healthcare data.



\## Overview



This project was developed as a visual analytical artifact inspired by the article:



\*\*“A Data-Driven Simulation Framework for Hospital Bed Management Using Synthetic Data”\*\*



The dashboard supports exploratory monitoring of hospital operational indicators, focusing on:



\* bed occupancy rates;

\* adult ICU occupancy;

\* exploratory forecasting;

\* data quality assessment;

\* Shewhart-inspired statistical monitoring;

\* executive visualization for decision support.

\## Installation



Clone the repository:



```bash

git clone https://github.com/anonimo-dashboard/dashboard-hospital-bed-management.git

```



Access the project folder:



```bash

cd dashboard-hospital-bed-management

```



Install dependencies:



```bash

pip install -r requirements.txt

```



Run the application:



```bash

streamlit run app.py

```
